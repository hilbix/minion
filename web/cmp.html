please enable JavaScript
  • Visually compare 2 or 4 images
    • Drop images anywhere onto this window
      • Images dropped as files are displayed immediately
      • Images or selections from (other) browsers display the dropped HTML code in red for safety.
      • If the code might contain malicious things (like <iframe> which might expose sensitive data) hit the 'clear' button!
      • Only click on the code to expand it if you are really sure. If unsure save the image to a local file and drop it as file.
      • If 'automatic add image to selector' is unchecked, Click on the image to put it into the selectior
    • In the selection area
      • On the left, select the quadrant (1, 2, 3, 4) to put image to
      • On the right, click the image to put into that quadrant (if there are no images, drop them first)
    • Move the handle (green knob) or bars (yellow) to chose the split view of the images
    • Works best with images of the same size
  • Adjustments:
    • wh: size of the compare area
    • hand: position of the separation (AKA green knob)
    • size: borders and handle
  • Following is not yet implemented (AKA bugs):
    • Data is not saved into Browser's SessionStore, so everything is lost on reload!
    • Holding down the mouse button while moving handle or bar is impossible if size is reduced to 0,0
    • Automated/Random sliding of yellow bars is missing
    • Zoom (to individually resize the images)
    • Pan (to individually position the images)
    • Hash parameters to preset everything / History / Data save/load
    • Keyboard navigation
    • File selector
    • changeable background canvas (for transparent images etc.), possibly animated
