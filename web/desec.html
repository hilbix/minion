  • This is a little frontend for desec.io to manage Tokens and Domains
    • To use it you need to login to desec.io
    • generate a token with https://desec.io/tokens by clicking on the (+)
    • optinally allow it to manage tokens
    • If you are paranoid, limit it to your IP (see it with an external service)
  • It does not transmit data to a thrird party:
    • All of your valuable data (like the token) is kept in your browser only
    • No data is transmitted to the server which serves these pages
    • Communication is done with desec.io servers only
    • But do not trust me! Get a good firewall which restricts these pages to desec.io!
    • Sadly, I do not know any and I doubt, it can be implemented with Manifest v3 from Google Chrome at all.
    • Read: AFAICS Google destroyed all possible network safety with Manifest v3
    • Read: Switch to Firefox or (perhaps) other browsers which allow to protect your data.
    • BTW I am working on a basic firewall extension for FireFox.
